# Shadow Steps

Shadow Steps is a React + Vite progressive web app starter. It includes an installable manifest, service worker caching, an offline fallback page, persistent local state, and Netlify build settings.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.

## PWA Features

- Web app manifest with standalone display mode.
- Service worker registration in production builds.
- Offline fallback page and cached app shell.
- Local progress persistence with `localStorage`.

## Deploying

This project includes `netlify.toml` with the Vite build command and publish directory configured for Netlify.
